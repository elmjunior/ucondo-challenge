# uCondo Challenge

Project developed in React Native for the Developer vacancy technical challenge

## Instructions

- Install the CLI: `npm i -g expo-cli`
- Install project dependencies: `yarn install`

### Run iOS

```
yarn ios
```

### Run Andoird

```
yarn android
```
